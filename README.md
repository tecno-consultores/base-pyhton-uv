# base-python-uv

A Python (Debian) Docker image with UV and RUFF. The tag corresponds to the python version

List of installed S.O. packages:

* build-essential
* gettext
* ca-certificates
* curl
* git
* zip
* unzip
* wget
* nano
* pipx

List of installed Python packages:

* uv
* ruff

Get this image on:
